## âš ï¸ Important Notes

### Why Archive?
Documents are archived when:
1. Work is complete and documented elsewhere
2. Historical record needed for reference
3. Implementation details superseded by newer approach
4. Context useful for understanding evolution

### Active Documentation
For current documentation, see:
- `/docs/guides/` - How-to guides
- `/docs/architecture/` - Current architecture
- `/docs/development/` - Development practices

### Don't Delete
Archive documents should NOT be deleted because:
- Provide historical context
- Explain why decisions were made
- Show evolution of codebase
- Useful for retrospectives
